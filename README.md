# ⌚ Timeless Watches Store

A modern and responsive luxury watch product page built using **HTML, CSS, and JavaScript**.
This project showcases a clean UI design for an e-commerce product page.

---

## 🚀 Live Demo

👉  https://vikaspawar-dev.github.io/timeless-watches-store/

---

## 📌 Features

* 🖼️ Product Image Gallery (Click to change image)
* 🔢 Quantity Selector (Increase / Decrease)
* 🛒 Add to Cart Button (Interactive)
* 🔍 Search Bar (Basic functionality)
* ⭐ Product Ratings Section
* 💻 Fully Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📁 Project Structure

```
timeless-watches-store/

```

## ⚙️ How to Run Locally

1. Download or clone the repository
2. Open `index.html` in your browser
